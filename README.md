### range()
- 이터러블 객체 생성
```python
range(n) # 0 이상 n 미만인 수를 나열하는 수열 생
range(a, b) # a 이상 b 미만인 수를 차례로 나열하는 수열
range(a, b, step) # a 이상 b 미만인 수를 step 간격으로 나열하는 수열
```

### 이터러블
- 반복할 수 있는 객체로, for~in 문에 사용할 수 있다.
- 이터러블 자료형 : list, str, tuple
